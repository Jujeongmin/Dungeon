# 🧟 Dungeon (좀비고 스타일 로그라이크)

## 📖 프로젝트 소개

좀비고와 유사한 탑다운 시점의 로그라이크 액션 게임을 1인 개발한 프로젝트입니다.
웨이브 기반 전투와 성장 요소를 결합하여 반복 플레이가 가능하도록 게임 루프를 설계했습니다.

---

## 🛠 개발 정보

* 개발 인원 : 1인 개발
* 개발 기간 : 2025.12 ~ 2026.02
* 엔진 : Unity (2D)
* 플랫폼 : PC, Web, Mobile

---

## ⭐ 주요 기능

### 웨이브 기반 적 스폰 시스템

스테이지 진행도에 따라 적이 주기적으로 생성되며
웨이브 증가에 따라 적 능력치가 상승하도록 설계했습니다.

### 플레이어 전투 시스템

이동·공격 등 핵심 전투 로직을 구현했습니다.

### 성장 시스템

플레이가 반복될수록 오직 플레이어의 실력만 향상되도록
로그라이크 성장 구조를 구현했습니다.

### UI 및 전투 피드백

공격, 피격 이펙트 연동으로
전투 몰입도를 강화했습니다.

---

## ⚙️ 기술 스택

* Unity 2D
* C#
* Animator
* Sprite Library / Sprite Resolver

---

## 🧩 트러블 슈팅

### 이동 애니메이션 성능 저하

방향별 애니메이션을 Animator 클립으로 관리하면서
상태 전환 비용과 드로우콜이 증가하는 문제가 발생했습니다.

Sprite Library · Resolver 구조를 도입하여
동일 타임라인에서 스프라이트만 교체하도록 개선했고,
그 결과 프레임 드랍을 완화하고 메모리 사용량을 절감했습니다.
